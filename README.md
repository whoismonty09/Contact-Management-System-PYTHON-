# Contact Management System (Python)

## Description
This project is a simple Contact Management System built using Python. It allows users to add, view, search, and delete contacts using a menu-driven command-line interface. The contact data is stored in a text file, making the information persistent even after the program is closed.

## Features
- Add new contacts (Name, Phone Number, Email)
- View all saved contacts
- Search contact by name
- Delete a contact
- Data stored in a file (contacts.txt)
- Menu-driven command-line interface

## Technologies Used
- Python
- File handling
